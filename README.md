# Xdits Media Website

A premium, modern, and interactive portfolio website for Xdits Media—a professional video editing agency specializing in story-driven content for creators and brands.

## Features

- **Responsive Design:** Fully optimized for desktop, tablet, and mobile browsers.
- **Three.js Animated Background:** Custom comet animation for a dynamic, premium feel.
- **Microinteractions:** Smooth, mobile-friendly transitions and feedback for buttons, cards, forms, and more.
- **Portfolio Showcases:** Interactive work section with embedded video reels and project types.
- **Modern UI/UX:** Custom cursor, animated cards, and scroll-triggered effects.
- **Contact Form:** Integrated with EmailJS for direct inquiries.
- **Social & WhatsApp Integration:** Easy access to social profiles and instant chat.

## Tech Stack

- **HTML5 & CSS3** (with custom variables and responsive breakpoints)
- **JavaScript (ES6+)**
- **Three.js** for 3D background animation
- **GSAP** for advanced animations and scroll effects
- **EmailJS** for contact form handling

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   ```
2. **Open the project folder:**
   ```bash
   cd XditsMedia
   ```
3. **Open `index.html` in your browser.**

No build step is required—everything runs client-side.

## Customization
- Update images, videos, and icons in the `assets/` folder.
- Edit text and content in `index.html` and portfolio HTML files.
- Adjust styles in `css/main.css` for branding and color changes.
- Modify Three.js animation in `js/background.js` for a unique background effect.
